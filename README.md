## Description

Aplikasi pengelolaan data tugas berbasis java yang ditargetkan untuk mahasiswa yang sulit membagi waktu untuk mengerjakan tugas yang dibarengi dengan
kewajiban di luar jam perkuliahan, tak hanya itu aplikasi ini juga memiliki berbagai fitur untuk membantu mahasiswa dalam hal lainnya.

### Features and It's Function

- Melihat Daftar Tugas
  Fitur ini berguna untuk memberikan informasi dari tugas yang ada, beserta info deadline tugas,
  tingkat prioritas tugas, status pengerjaan tugas, serta sisa hari sebelum deadline tugas.
- Deadline terdekat
  Fitur ini berguna untuk memberikan informasi susunan tugas berdasarkan deadline terdekat.
- Tampilan Tugas Kelompok
  Fitur ini berguna untuk memberikan informasi daftar tugas kelompok yang berisikan informasi tugas beserta nama anggota kelompok.
- Menambahkan Tugas Baru
  Fitur yang berguna untuk menambahkan  tugas baru beserta tingkat prioritas tugas, deadline tugas, serta mata kuliahnya.
- Menghapus Tugas
  Fitur ini berguna untuk menghapus tugas yang diinginkan berdasarkan tampilan daftar tugas.
- Mengedit Tugas
  Ubah Status Tugas
  Fitur ini berguna untuk mengubah status pengerjaan tugas, antara sudah selesai atau belum selesai.
- Mata Kuliah
  Fitur yang membolehkan pengguna untuk memanajemen mata kuliah yang ditempuh, dapat digunakan untuk mengelompokkan tugas, catatan, dan link.
- Catatan Mata Kuliah
  Manajemen catatan tambahan dari mata kuliah bilamana pengguna mendapat ilmu tambahan di luar materi yang sudah diajarkan.
- Link Pembelajaran Mata Kuliah
  Manajemen data link yang dapat berisikan suatu media (seperti link youtube ataupun dokumen),
  yang berkaitan langsung dengan mata kuliah yang sudah ditambahkan dalam aplikasi.
- Notifikasi
